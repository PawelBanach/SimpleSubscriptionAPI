# SimpleSubscriptionApi

Project setup:

1. `bundle`
2. `rake db:setup`
3. `raek db:migrate`
4. `rails c`

Run tests (_few tests still todo_):

`bundle exec rspec`

Code coverage:

`http://localhost:63342/SimpleSubscriptionAPI/coverage/index.html`

or in folder:

`coverage/index.html`

Static code analyzer:

`rubocop`

`rails_best_practices .`
